# eb (Einbürgerungstest App)

I created this app to test me on the most difficult questions of the Einbürgerungstest, useful and easy-to-use on a smartphone. This app only has most difficult 54 of the 300 questions possible on the test, since most of the 300 questions on the test are quite simple. I reviewed all 300 questions and the ones I got wrong or found tricky, I saved in a text file and used Antigravity/Gemini for a couple hours to build this app.
